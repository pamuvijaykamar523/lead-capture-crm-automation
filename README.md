An automated Lead Capture CRM workflow built with n8n, Airtable, Gmail, and Webhooks.
# Lead Capture CRM Automation

## Overview

This project automates the process of capturing leads from a web form and storing them in Airtable while sending an instant email notification.

## Workflow

HTML Form
⬇
Webhook (n8n)
⬇
Airtable CRM
⬇
Gmail Notification
⬇
Webhook Response

## Features

- Capture lead details
- Receive data using Webhooks
- Store records in Airtable
- Send automatic email notifications
- Return success response to users

## Tech Stack

- n8n
- Webhooks
- Airtable
- Gmail
- HTML
- REST API
